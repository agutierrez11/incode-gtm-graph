## 🇺🇸 English

### Project Overview
This repository contains the interactive knowledge graph mapping the LATAM fintech ecosystem, regional compliance regulations, and primary competitors in the identity verification and biometrics market (such as Sumsub, MetaMap, Truora, and LexisNexis) for **Incode Technologies**.

The visualization is built using **Cosma**, modelizing interlinked markdown documents into a searchable, navigable semantic web interface (Cosmoscope).

### Visual Architecture & Node Coloring
Nodes in the network represent specific research entities categorized and colored by type:
* 🟣 **Strategy (`Estrategia`):** Go-To-Market plans, value propositions, and targeting frameworks.
* 🟢 **Products (`Productos`):** Core product modules (e.g., Liveness Detection, KYC/KYB, Document OCR, Edge/SaaS deployments).
* 🟡 **Competitors (`Competencia`):** Direct and indirect regional competitors, highlighting strengths and weaknesses.

### How to use the Interactive Map
1. Click the **Live Interactive Graph Link** above.
2. Search for any term or company (e.g., "KYC", "Sumsub", "Mexico") in the search bar.
3. Click on any node to view its connections and read the detailed research notes in the sidebar.
4. Drag and zoom the graph nodes to explore how compliance regulations, products, and competitive targets are interconnected.

## 🇪🇸 Español

### Descripción del Proyecto
Este repositorio contiene el mapa de conocimiento interactivo que analiza el ecosistema fintech en LATAM, las regulaciones de cumplimiento regional y a los principales competidores del mercado de verificación de identidad y biometría (como Sumsub, MetaMap, Truora y LexisNexis) para **Incode Technologies**.

La visualización está construida con **Cosma**, modelando documentos Markdown interconectados en una interfaz web semántica (Cosmoscope) que permite búsquedas y navegación fluida.

### Arquitectura Visual y Colores de Nodos
Los nodos en la red representan entidades de investigación específicas organizadas y coloreadas por tipo:
* 🟣 **Estrategia:** Planes de Go-To-Market, propuestas de valor y marcos de segmentación de clientes.
* 🟢 **Productos:** Módulos de producto principales de Incode (ej. Detección de Liveness, KYC/KYB, OCR de Documentos, despliegues Edge/SaaS).
* 🟡 **Competencia:** Competidores regionales directos e indirectos, destacando fortalezas y debilidades.

### Cómo usar el Mapa Interactivo
1. Haz clic en el enlace del **Mapa Interactivo en Vivo** que aparece arriba.
2. Busca cualquier término o empresa (ej. "KYC", "Sumsub", "México") en la barra de búsqueda.
3. Haz clic en cualquier nodo para ver sus conexiones y leer las notas de investigación detalladas en el panel lateral.
4. Arrastra y haz zoom en los nodos del grafo para explorar cómo se interconectan las regulaciones, los productos y los objetivos competitivos.
